# 🚬 smoking-biomarker-analysis

건강검진 데이터(7,000건, 18개 변수)를 활용해 흡연 여부와 생체 지표 간의 상관관계를 분석한 EDA 프로젝트.

## 결론

초기 가설(LDL)은 BMI 교란변수로 인해 기각됐고, **중성지방(TG)** 이 흡연을 가장 순수하게 반영하는 핵심 지표로 도출됐다.
중성지방을 로지스틱 회귀 모델에 추가하자 설명력이 54% 향상됐다 (Pseudo R² 0.050 → 0.077).

## 사용 기술

`Python` `Pandas` `Seaborn` `statsmodels`

## 파일 구조

```
├── 1_1_흡연여부데이터_정윤아.ipynb   # 개인 EDA (데이터 정제, 단변량 분석)
├── 2조_미니프로젝트_최종.ipynb       # 팀 최종본 (이변량 분석, 로지스틱 회귀)
└── 2조_미니프로젝트_발표.pdf         # 발표 자료
```

데이터는 교육 과정 제공 자료로 외부 공개가 제한됨.
